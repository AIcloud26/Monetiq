# Deployment

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/monetiq.git
```

## 2. Open Project

```bash
cd monetiq
```

## 3. Push to GitHub

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

## 4. Enable GitHub Pages

Go to:

GitHub Repository → Settings → Pages

Then select:

```txt
Deploy from branch
main
/root
```

Save.

Your site will be available at:

```txt
https://yourusername.github.io/monetiq/
```
